# tab-player-cli

Write string instrument tabs in a plain text file and play them in the command line!

Empty template example:

```

    |           |           |           |           |           |           |           |

G3 |-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|
D3 |-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|
A2 |-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|
E2 |-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|
```

```
   1/4         2/4         3/4         4/4
    |           |           |           |           |           |           |           |
E4 |-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --|

```
then run `tab-player-cli --play <TAB_FILE> <SAMPLES_FOLDER>`
